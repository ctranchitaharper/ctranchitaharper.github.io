I had learned a few things, such as what IPSec and SSL was as well as the different types of threats that you can encounter when browsing the internet. For example, IPSec is an entire set of protocols that provides security at a network level. So that means it does not provide any security on your local machine, but instead provides security on your network or internet. I also learned that SSL provides security on the transport layer. This includes privacy, integrity, and authentication. This means that for someone to access your internet, they need to first be authenticated using digital certificates and asymmetric cryptography. This allows you to keep your privacy as well as your network and computer safe from threats. I can use this in the real world by ensuring that my own as well as others SSL hasn't been breached when trying to diagnose potential issues.
