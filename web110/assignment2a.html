The two activities I chose were install multiple web browsers to compare the different programs and configure browser settings. When I had completed the first activity I had learned that Chrome and OperaGX's download and install wizards weren't that different. Everything, for the most part, that chrome had in it's wizard so did OperaGX. I realized that due to there being a standard that consumers hold a company to most of all browsers download and install process will be very similar, if not the same, to each other. I can use this in my own environment by not mulling over which browser has better a download and install process than the other. It won't dictate whether or not I use one browser over the other. Next, I completed the second activity. the biggest difference between Google Chrome (Chrome) and OperaGX was that Chrome was more of a streamlined web browser while OperaGX was more so for enthusiasts. Chrome also didn't have the customization capability that OperaGX has. The settings where similar due to OperaGX using Chrome as a base. A lot of the same extensions and settings that Chrome has OperaGX also has. I can apply these concepts in many ways. For example, I value OperaGXs customization capabilities and tools so I personally would use OperaGX over Chrome.
